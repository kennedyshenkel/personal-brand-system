# Starter Kit for Claude.ai Projects

This is a simplified version of the Personal Brand Content System for users who don't use Claude Code. It works with Claude.ai Projects (web interface).

## What You Get

- A guided voice discovery interview
- Anti-AI-slop quality rules
- Writing that sounds like you, not AI
- LinkedIn-optimized content guidance

## What You Don't Get (Claude Code Only)

- The full `/setup` modular interview with progress tracking
- Persistent vault with file management
- 9 specialized skills (/brainstorm, /research, /voice-check, etc.)
- Automated content index tracking
- `/refine` for iterative voice improvement

## Setup Instructions

1. Go to [claude.ai](https://claude.ai) and create a new **Project**
2. Open the Project's **Custom Instructions**
3. Copy the entire contents of `system-prompt.md` and paste it as the custom instructions
4. Upload `anti-slop-rules.md` as a **Project File** (drag and drop or use the upload button)
5. Start a new conversation in the Project
6. Claude will begin the voice discovery interview automatically

## After Setup

Once Claude has interviewed you, it will produce a brand profile and voice reference. Copy these into the Project's custom instructions or save them as additional Project Files so they persist across conversations.

## Upgrading to Claude Code

For the full experience (all 9 skills, persistent vault, progress tracking, automated workflows), use the main repository with Claude Code. See the root README.md for instructions.
