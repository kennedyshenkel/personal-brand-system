# Personal Brand Content System

A Claude Code project that builds and operates your personal brand content pipeline. It discovers your writing voice, plans your content strategy, writes in your style, audits quality, and manages your output across platforms.

Built for anyone who wants to build a personal brand with writing that sounds like them, not like AI.

## What This Does

1. **Interviews you** to learn your voice, expertise, values, and audience
2. **Generates a brand profile and voice style guide** tailored to how you actually write
3. **Writes content** that sounds like you at your best, across platforms and registers
4. **Enforces quality** with hardcoded anti-AI-slop rules that prevent generic, corporate, or AI-sounding output
5. **Plans your content strategy** with platform-specific cadence, format, and topic recommendations
6. **Improves over time** as you write more and refine your profile

## Prerequisites

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed (CLI, desktop app, or web at claude.ai/code)
- A Claude account (Pro or Team plan)
- 20-40 minutes for initial setup

## Quick Start

1. Click **"Use this template"** on GitHub to create your own repository
2. Clone your new repository locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
   cd YOUR-REPO-NAME
   ```
3. Open it in Claude Code:
   ```bash
   claude
   ```
4. Run `/setup` to build your brand profile and discover your voice

## What Happens During Setup

The `/setup` skill runs a modular interview across 9 sections:

| Module | What It Covers | Required? |
|--------|---------------|-----------|
| Identity | Name, role, career history, differentiator, brand purpose | Yes |
| Voice Discovery | Writing samples analysis OR guided voice exercises | Yes |
| Content Pillars | 3-6 topics where you have genuine authority | Yes |
| Audience | Who you're writing for (professional + personal) | No |
| Platforms | Where you'll publish (LinkedIn is default) | No |
| Content Boundaries | Topics that are off-limits or restricted | No |
| Interests | Personal interests that inform your content | No |
| Creators | Thought leaders you follow and engage with | No |
| Rhetorical Techniques | Signature writing moves personalized to your voice | No |

After completing Identity, Voice Discovery, and Content Pillars, your content skills unlock. You can complete the remaining modules anytime.

**Have existing writing?** Provide 3-5 samples during Voice Discovery for the strongest voice profile. **No writing yet?** The guided exercise path works too. Write 3-5 pieces, then run `/refine` to sharpen your profile.

## Customization

### Adding Platforms
Run `/setup` and complete the Platforms module, or edit `brand-profile.md` directly.

### Adding Skills
Create a new directory in `.claude/skills/`, add a `SKILL.md` file with YAML frontmatter, and register it in the Available Skills table in `CLAUDE.md`.

### Updating LinkedIn Benchmark Data
The LinkedIn playbook (`linkedin-playbook.md`) contains sourced benchmark data from 2025-2026. When new data is available, update the playbook and its AI-facing reference (`linkedin-playbook-reference.md`).

## FAQ

**"I don't have writing samples."**
The guided voice discovery path in `/setup` works without samples. You'll answer questions about your communication style, what writing you love/hate, how you express disagreement, etc. After writing 3-5 pieces, run `/refine` to sharpen your profile with real output.

**"Can I use this for a company brand?"**
It's designed for personal brands, but the architecture works for any voice-driven content system. You'd need to adjust the brand profile sections and potentially the register modulation.

**"How do I update my voice profile?"**
Run `/refine`. It reads what you've written, compares it to your current voice reference, and proposes targeted updates.

**"What are the anti-slop rules?"**
A hardcoded set of rules that prevent writing from sounding AI-generated: banned phrases ("delve," "In today's..."), banned formatting (em dashes), banned structural patterns (intro-body-conclusion that restates intro), and quality standards (active voice, evidence-backed claims, sentence length variation). These apply to all content and can't be overridden.
